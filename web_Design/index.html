Match-day memories and my love for Manchester United.

Welcome to my football journal. This website documents my experience at Emirates Stadium on 15 November and explains why Manchester United is my favourite club. I built the site using HTML, CSS, and JavaScript with a focus on clean design, responsive layout for different screen sizes, and accessibility.

The site has four pages and each page has a clear purpose. The About page is mainly text-based to demonstrate typography and readable spacing. The Gallery page focuses on image presentation with captions and a responsive grid. The Magazine page combines text and images in a mixed layout to create a feature-article style design.
